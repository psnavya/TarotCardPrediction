# TarotCardPrediction

This Spring Boot application that can be used to draw a tarot card from a deck.
For details, have a look at https://medium.com/@psnavya90/basic-spring-boot-code-snippet-to-draw-a-tarot-card-from-deck-6aa396314379

**Execution:**
Run the Spring Boot application. You can use your IDE to run the main method in the TarotApplication class.

Access the tarot card drawing endpoint by navigating to http://localhost:8080/tarot/draw in your browser. You should see a JSON response with the name and description of a randomly drawn tarot card.
